## Install dependencies


## Run the application

1- Run backend

```sh
cd backend
```

```sh
yarn
```

```sh
yarn start
```

2- Run Frontend Local

In the root directory make sure to copy env_template to .env and fill the values then run:

```sh
yarn start
```

## Run tests
